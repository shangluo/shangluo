Leetcode-Serialize and Deserialize Binary Tree

没有仔细研究Leetcode自己的serialize方式，这里是直接用括号来划分层次:
例如：

    1
   / \
  2   3
 /
21

会被序列化为(1(2(21)())(3))
加上缩进后层次化更明显：

(1    // root node
    (2
          (21)
          () // 空节点
    )
    (3)
 )

序列化很简单，中序遍历，递归输出对应节点即可。
解析的时候，遇到 ‘(‘ 递归调用deserialize，最终生成完整二叉树。
